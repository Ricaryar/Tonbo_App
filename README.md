# 瞳伴 (Tonbo) - 智能視覺助手

瞳伴是一個專為視障人士設計的智能助手應用，結合計算機視覺技術和語音提示，幫助用戶在日常生活中進行環境感知、文件閱讀和緊急求助。

**倉庫地址**：[github.com/Charlieppy2/NEW_Tonbo](https://github.com/Charlieppy2/NEW_Tonbo)

## 🔌 完全離線能力（v3.0）

本版本支持在**無網絡**環境下使用粵語語音命令與智能對話：

| 模塊 | 技術 | 大小 | 說明 |
|------|------|------|------|
| **離線語音識別 (ASR)** | Sherpa-ONNX Paraformer | ~238 MB | 粵語 / 普通話 / 英文三語流式識別 |
| **離線大語言模型 (LLM)** | Qwen2 1.5B + LoRA (GGUF q8_0) | ~1.6 GB | 自訓練粵語對話模型，完全本地推理 |
| **物體檢測** | SSD MobileNet / YOLOv8 (TFLite) | ~20 MB | 離線環境識別 |
| **語音合成 (TTS)** | Android 系統 TTS | — | 使用設備內建語音引擎 |

首次啟動時，APK 內嵌模型會自動解壓到應用內部存儲，無需手動配置。

### 離線模型路徑（手機端）

```
/data/data/com.example.tonbo_app/files/llm/qwen-tonbo-q8_0.gguf
/data/data/com.example.tonbo_app/files/sherpa_asr/sherpa-onnx-streaming-paraformer-trilingual-zh-cantonese-en/
```

### 離線回退策略

- **ASR**：優先 Sherpa-ONNX → 不可用時回退 Google 在線識別
- **LLM**：優先本地 GGUF → 不可用時回退 DeepSeek / GLM-4-Flash 在線 API → 最後回退關鍵詞匹配

### **在線 LLM（回退）**
- **DeepSeek** ⭐ (默認): 每月 100 萬 tokens 免費額度
- **GLM-4-Flash**: 完全免費，中文支持優秀

### **回應優先級**
```
離線 GGUF → 在線 DeepSeek / GLM-4-Flash → 關鍵詞匹配
```

本倉庫通過 **Git LFS** 管理大文件（約 6.5 GB），克隆前請先安裝 [Git LFS](https://git-lfs.github.com/)：

```powershell
git lfs install
git clone https://github.com/Charlieppy2/NEW_Tonbo.git
cd NEW_Tonbo
git lfs pull
```

> **注意**：GitHub 免費賬戶 LFS 配額有限（約 1 GB 存儲 + 1 GB/月流量）。若 `git lfs pull` 失敗，可在 GitHub 設置中購買 LFS Data Pack，或自行下載模型後放入對應目錄。

```
app/src/main/assets/

training/output/sherpa_asr/             # Sherpa 完整模型包（含 tar.bz2）
releases/

└── tonbo-offline-debug.apk.part002
```

完整 APK 約 2.7 GB，已拆成兩個分片上傳。合併方法：

```powershell
.\training\merge_apk_parts.ps1
# 輸出：releases/tonbo-offline-debug.apk
```

- Android Studio Arctic Fox 或更新版本
- Android SDK API 21+
- Gradle 8.0+

### 首次設置（重要）

**在不同電腦上使用時，需要配置 Android SDK 路徑：**

1. **方法一：創建 `local.properties` 文件（推薦）**
   ```bash
   # 複製模板文件
   cp local.properties.template local.properties
   
   # 編輯 local.properties，設置你的 Android SDK 路徑
   # macOS: sdk.dir=/Users/YOUR_USERNAME/Library/Android/sdk
   # Linux: sdk.dir=/home/YOUR_USERNAME/Android/Sdk
   # Windows: sdk.dir=C:\\Users\\YOUR_USERNAME\\AppData\\Local\\Android\\Sdk
   ```

2. **方法二：設置環境變量**
   ```bash
   # macOS/Linux
   export ANDROID_HOME=/path/to/your/android/sdk
   
   # Windows (PowerShell)
   $env:ANDROID_HOME="C:\Users\YOUR_USERNAME\AppData\Local\Android\Sdk"
   ```

**注意**：`local.properties` 文件已被 `.gitignore` 排除，不會提交到 Git，每個開發者需要在自己的電腦上創建。

可選：在 `local.properties` 中覆蓋模型路徑（用於開發部署腳本）：
```properties
tonbo.gguf.path=D:\\path\\to\\qwen-tonbo-q8_0.gguf
tonbo.sherpa.dir=D:\\path\\to\\sherpa-onnx-streaming-paraformer-trilingual-zh-cantonese-en
```

```powershell
.\gradlew.bat :app:assembleDebug

./gradlew :app:assembleDebug
```
